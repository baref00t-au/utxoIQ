# Keyboard Shortcuts Guide

## Overview

utxoIQ provides comprehensive keyboard shortcuts to help you navigate and interact with the platform efficiently. This guide covers all available shortcuts, organized by context and functionality.

## Quick Reference

Press **?** anywhere in the application to display the keyboard shortcuts help modal.

## Global Shortcuts

These shortcuts work throughout the entire application:

| Shortcut | Action | Description |
|----------|--------|-------------|
| **?** | Show shortcuts help | Display keyboard shortcuts modal |
| **/** | Focus search | Jump to search input |
| **Escape** | Close/Cancel | Close modals, dialogs, or cancel actions |
| **Ctrl/Cmd + K** | Command palette | Open quick command menu |
| **Ctrl/Cmd + /** | Toggle theme | Switch between dark and light themes |

## Platform-Specific Shortcuts

### Windows/Linux

- **Ctrl** is the primary modifier key
- **Alt** for secondary shortcuts
- **Shift** for extending actions

### macOS

- **Cmd (⌘)** is the primary modifier key
- **Option (⌥)** for secondary shortcuts
- **Shift (⇧)** for extending actions
- **Control (⌃)** for specific actions

## Customizing Shortcuts

### Viewing Current Shortcuts

1. Press **?** to open shortcuts help
2. Browse by category
3. Search for specific shortcuts

### Changing Shortcuts (Power Tier)

1. Navigate to **Settings → Keyboard Shortcuts**
2. Click shortcut to change
3. Press new key combination
4. Click **Save** to apply

### Resetting Shortcuts

1. Navigate to **Settings → Keyboard Shortcuts**
2. Click **"Reset to Defaults"**
3. Confirm reset
4. All shortcuts restore to defaults

## Shortcut Conflicts

### Avoiding Conflicts

The application detects and prevents:
- Conflicts with browser shortcuts
- Conflicts with OS shortcuts
- Conflicts between application shortcuts

### Resolving Conflicts

If a shortcut doesn't work:
1. Check for browser extension conflicts
2. Verify OS-level shortcuts
3. Try alternative shortcut
4. Customize conflicting shortcut

## Tips and Tricks

### Learning Shortcuts

1. **Start with basics**: Learn navigation shortcuts first
2. **Use help modal**: Press **?** frequently
3. **Practice daily**: Use shortcuts in regular workflow
4. **Customize**: Adjust shortcuts to your preferences
5. **Share**: Teach shortcuts to team members

### Productivity Workflows

#### Quick Insight Review
```
1. Press G then F (go to feed)
2. Press J/K to navigate insights
3. Press O to open interesting insights
4. Press B to bookmark important ones
5. Press E to export results
```

#### Dashboard Setup
```
1. Press G then D (go to dashboard)
2. Press Ctrl/Cmd + A (add widgets)
3. Use Tab to navigate widget library
4. Press Enter to add widgets
5. Press Ctrl/Cmd + S (save layout)
```

#### Filter and Export
```
1. Press / (focus search)
2. Type search query
3. Press Ctrl/Cmd + 1-9 (apply preset)
4. Press E (export results)
5. Choose format and download
```

## Troubleshooting

### Shortcut Not Working

**Problem**: Keyboard shortcut doesn't respond

**Solutions**:
1. Check if element has focus
2. Verify shortcut is correct
3. Check for conflicts
4. Try refreshing page
5. Check browser console

### Wrong Action Triggered

**Problem**: Shortcut triggers unexpected action

**Solutions**:
1. Verify correct key combination
2. Check for modifier keys
3. Review shortcut customizations
4. Reset to defaults
5. Report bug if persistent

### Shortcuts Disabled

**Problem**: No shortcuts work

**Solutions**:
1. Check accessibility settings
2. Verify JavaScript enabled
3. Check browser extensions
4. Try different browser
5. Contact support
