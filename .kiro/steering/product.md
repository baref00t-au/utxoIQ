# Product Overview

utxoIQ is an AI-powered Bitcoin blockchain intelligence platform that transforms raw on-chain data into human-readable insights for traders, analysts, and researchers.

## Core Value Proposition

- **Real-time Intelligence**: Processes Bitcoin blocks within 60 seconds to generate actionable insights
- **AI-Powered Analysis**: Uses Vertex AI to explain not just what happened, but why it matters
- **Evidence-Based**: All insights include blockchain citations (block heights, transaction IDs)
- **Multi-Channel Distribution**: Web platform, API, and automated social media posting

## Key Features

- **Insight Feed**: Chronological display of AI-generated blockchain insights with confidence scores
- **Daily Brief**: Curated summary of top 3-5 blockchain events published at 7 AM UTC
- **Custom Alerts**: User-defined thresholds for mempool fees, exchange flows, and miner activity
- **AI Chat**: Natural language queries about Bitcoin blockchain data
- **Social Integration**: Automated X (Twitter) posting of high-confidence insights

## Target Users

- Bitcoin traders seeking real-time market intelligence
- Crypto analysts requiring daily blockchain summaries
- Researchers monitoring specific blockchain metrics
- Blockchain enthusiasts wanting accessible data insights

## Subscription Tiers

- **Free**: Basic insight access with limited features
- **Pro**: Enhanced alerts and extended API access
- **Power**: Unlimited chat queries and advanced customization
